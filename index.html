How Do I Install Garmin Express?
Garmin Express is an application designed to manage Garmin devices. It is used for device registration, map updates and software updates, syncing Fitness data with Garmin Connect, and much more.

NOTE: Garmin Express is not compatible with tablets or phones and must be used with a computer that features a USB port for a data cable or ANT+ Stick to communicate. For more details see Can I use Garmin Express on a Phone or a Tablet?.

INSTALL GARMIN.COM./EXPRESS ON MY COMPUTER?

Garmin Express is designed to be the one-stop shop for updating maps, updating software, managing favorites, registering a device, and much more.

To download the latest version of Garmin Express:

Windows XP SP3 or newer

  1. Go to: http://garmin.com/express
  2. Click Download for Windows
  3. Run GarminExpress.exe to start installation
  4. Check the box next to the terms and conditions statement
  5. Advanced Users can select the Options button to change the install location of Garmin Express if needed
  6. Press the Install button
  7. Press the Launch button to open Garmin Express

Mac OS 10.7 or higher

  1. Go to: http://garmin.com/express
  2. Click Download for Mac
  3. Open GarminExpress.dmg from the Downloads folder or download location
  4. In the new window, double click on the Install Garmin Express icon
  5. Follow instructions in the installer
  6. Once install is complete, close the installer
  7. Eject the Garmin Express mounted drive from Finder or the Desktop
  8. Open the Applications folder
  9. Double click on the Garmin Express icon

Garmin Express is now installed and can be used with compatible GPS devices.
